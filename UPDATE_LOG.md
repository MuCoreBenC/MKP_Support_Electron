# 🚀 支撑面改善工具 (MKP Support) 迭代日志

> **项目目标**：为 3D 打印爱好者提供好用的支撑面优化工具。

---

## [v0.2.1] - 2026-03-10 (最新发布版本)
### 🎨 交互体验优化
- **平滑的复制动效**：重构了脚本路径复制按钮的交互。点击后按钮平滑展开并伴随状态颜色的交叉淡入淡出，消除了原先图标切换时的闪烁感，提供了更自然的视觉反馈。
- **非线性滚动展示**：优化了校准页面超长路径文本的展示方式。进入页面时，长路径会以非线性减速平滑滚动至末尾文件名区域，并修复了频繁切换页面可能导致的滚动冲突或留白异常。
- **单选框视觉净化**：优化了单选组件的焦点响应逻辑。使用鼠标点击时不再显示系统默认的外部高亮光环，仅保留点选动画，保持界面清爽；同时保留了针对键盘导航的无障碍提示。
- **全局底部留白**：改进了基础页面的排版引擎，为所有可滚动区域统一增加了底部安全边距，避免折叠面板展开或内容较多时，UI 元素紧贴窗口底边。

### ✨ 核心逻辑与稳定性
- **离线优先加载策略**：解除了本地预设列表对网络清单的依赖。本地配置的读取与渲染不再受网络状态影响，仅在用户手动点击“检查更新”时才发起网络请求，提升了软件启动速度与断网环境下的可用性。
- **准确的网络异常提示**：完善了云端同步时的异常捕获机制。当请求失败时，界面会直接显示具体的错误原因（如网络连接失败、云端文件不存在、格式解析异常等），替代原有的通用缺省提示，便于快速定位问题。

### 🛠️ 日志与排障
- **关键交互行为记录**：在本地日志系统中补充了“页面导航切换”、“主动检查更新”、“参数还原默认”、“XY轴坐标点击”等核心操作的记录，便于在出现问题时准确还原操作路径。

## [v0.2.0] - 2026-03-10 
### ✨ 核心功能
- **Z轴校准系统**：引入可视化 Z 轴偏移微调网格（-0.5 至 +0.5），替代原有的文本输入方式。
- **预设解析引擎优化**：重构了 JSON/TOML 文件的读取与写入逻辑，实现更准确的层级映射，优先保护底层 `toolhead.offset.z` 参数，防止数据错位。

### 💄 视觉与交互
- **Dock 栏模拟动效**：基于正弦波算法，为网格组件构建了物理挤压与悬浮放大的交互效果。
- **动效控制面板**：在设置页新增动效调节选项，支持自定义方块基础大小与放大系数，关闭总开关后相关控件会自动呈现灰度禁用状态。
- **克制的悬停反馈**：调整了网格的 CSS 渲染，未选中方块悬停时采用黑白反馈，仅对当前选中项注入主题色。
- **锚点平滑滚动**：优化了校准面板的展开体验，点击后页面自动平滑滚动并保留适当的顶部视距。

## [v0.1.9] - 2026-03-07
### ✨ 核心功能
- **全局弹窗引擎**：使用支持异步调用的自定义模态框（MKPModal）替换系统原生弹窗，支持多种提示状态。
- **固定表头布局**：通过指定属性锁定页面头部坐标，解决页面切换时标题与操作按钮的跳动问题。

### 💄 视觉与交互
- **微交互修复**：修复了部分表单控件的焦点样式问题，平滑了设置页开关组件的过渡动画。
- **按钮对齐优化**：调整了关键操作按钮的内部排版，确保文字与图标居中对齐。

## [v0.1.5] - [v0.1.8] - 2026-03-03 至 2026-03-06
### 🛠️ 工程化升级
- **消息提示系统**：封装全局小红点管理器，支持向特定元素动态挂载提示徽章。
- **自动更新机制**：接入 GitHub Releases，支持静默下载并在重启时自动完成软件更新。

### 🎨 布局重构
- **设置页导航**：实现设置页面的锚点跳转与滚动位置自动高亮联动。
- **外观模式优化**：重构暗黑模式变量逻辑，改善启动时的背景色同步表现。
