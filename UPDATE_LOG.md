# 🚀 支撑面改善工具 (MKP Support) 迭代日志

> **项目目标**：为 3D 打印爱好者提供最极致的支撑面优化交互体验。

---

## [v0.1.9] - 2026-03-07 (当前发布版本)
### ✨ 核心功能
- **SaaS 级全局弹窗引擎**：彻底告别系统原生弹窗，引入支持 Promise 异步调用的自定义模态框（MKPModal），支持 info, warning, error, success 四种状态。
- **全局布局引擎 2.0**：通过 `data-fixed-header` 属性锁死页面表头。标题、功能按钮、副标题坐标绝对对齐，彻底解决页面切换时的“跳动”感。

### 💄 视觉与交互
- **微交互调优**：修复单选按钮（Radio）的方形焦点 Bug，实现纯正圆环 SaaS 风格；将 Toggle 开关动画由“弹跳式”改为“线性平滑移动”。
- **按钮对齐策略**：重构了“下一步”等关键按钮的布局，确保文字与图标始终水平垂直居中。

---

## [v0.1.5] - [v0.1.8] - 2026-03-03 至 2026-03-06
### 🛠️ 工程化升级
- **全局小红点管理器**：封装 `RedDotManager` 函数，支持对任何 ID 元素一键挂载“呼吸动画”或“静止”状态的小红点，实现 APP 级的通知感知。
- **自动更新体系**：打通 GitHub Releases 链路，配置 `electron-updater` 实现静默下载与“重启立即更新”的闭环流程。
- **模型保护机制**：模型文件自动从只读目录释放至 `AppData/Roaming/mkp-support`，并支持调用系统关联程序（如 Bambu Studio）一键打开。

### 🎨 布局重构
- **网易云风格设置页**：实现顶部 Tabs 锚点跳转与滚动自动高亮高亮功能。
- **暗黑模式 2.0**：重构背景色逻辑，解决启动时侧边栏与主区域颜色同步迟缓的问题。

---

## [v0.1.2] - [v0.1.4] - 2026-03-01 至 2026-03-02
### ✨ 新增特性
- **动态色阶系统**：利用 `color-mix` 技术，实现了全局主色调的 50-950 色阶自动生成，确保 UI 细节处的色彩和谐。
- **版本专属图标色**：支持为标准版、快拆版、Lite 版自定义标识色，提升列表辨识度。

---

## [v0.0.1] - [v0.1.1] - 2026-02-27 至 2026-02-28
### 🌱 项目起步
- **核心框架搭建**：基于 Electron + Tailwind CSS 建立项目骨架。
- **数据驱动架构**：确立了由 `data.js` 驱动品牌、机型、预设列表的渲染逻辑。
- **基础功能**：实现后处理脚本路径自动生成、机型收藏、搜索过滤等基础功能。
